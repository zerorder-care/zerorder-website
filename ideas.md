# Zerorder 웹사이트 디자인 에셋 전략

## 프로젝트 개요
Zerorder(제로더)는 영아 사경(Infant Torticollis) 재활 솔루션 '바로도리'를 제공하는 헬스테크 스타트업입니다. 스타트업 프로그램(OpenAI, AWS, Notion 등) 지원을 위한 공식 웹사이트를 제작합니다.

## 디자인 철학 및 에셋 계획

### 1. 현대적 의료 기술 미니멀리즘 (Modern Medical Minimalism)
**확률: 0.08**

**설계 철학**: 의료 기술의 신뢰성과 현대성을 강조하면서도 부모들의 불안감을 덜어주는 따뜻한 감정을 전달합니다.

**핵심 원칙**:
- 깔끔한 화이트스페이스와 의도적인 여백으로 신뢰감 구축
- 의료 전문성과 기술 혁신의 균형
- 아기와 부모의 관계를 중심으로 한 인간중심 설계
- 과학 기반의 신뢰성 시각화

**색상 철학**:
- 주색: 따뜻한 청색(#0066CC) - 신뢰, 안정성, 의료 전문성
- 보조색: 부드러운 산호색(#FF6B6B) - 따뜻함, 케어, 아기의 건강
- 중립색: 밝은 회색(#F5F5F5) - 깔끔함, 현대성
- 악센트: 밝은 초록색(#00D084) - 성공, 회복, 치료 진행

**레이아웃 패러다임**:
- 비대칭 그리드: 좌측에 텍스트/정보, 우측에 시각적 요소
- 카드 기반 섹션 분리로 정보 계층화
- 수직 타임라인으로 사용자 여정 표현

**시그니처 요소**:
- 아기의 머리 움직임을 추상화한 곡선 모티프
- 부드러운 원형 일러스트레이션으로 아기와 부모 표현
- 의료 데이터 시각화(차트, 진행도)

**상호작용 철학**:
- 부드러운 호버 효과와 마이크로 인터랙션
- 스크롤 트리거 애니메이션으로 정보 공개
- 진행도 표시기로 사용자 여정 시각화

**애니메이션**:
- 페이드인/페이드아웃으로 부드러운 전환
- 스크롤 시 요소의 슬라이드인 효과
- 호버 시 미묘한 스케일 변화(1.02배)
- 로딩 상태에서 맥박 같은 애니메이션

**타이포그래피**:
- 디스플레이: Poppins Bold(700) - 헤딩, 강조
- 본문: Inter Regular(400) - 읽기 편한 기본 텍스트
- 강조: Inter SemiBold(600) - 중요한 정보

---

### 2. 따뜻한 인간중심 디자인 (Warm Human-Centered Design)
**확률: 0.07**

**설계 철학**: 부모의 불안감을 공감하고 해결책을 제시하는 따뜻하고 친근한 접근

**핵심 원칙**:
- 일러스트레이션 중심의 시각적 스토리텔링
- 아기와 부모의 감정에 집중
- 커뮤니티와 공유의 가치 강조
- 접근성과 이해하기 쉬운 정보 구조

**색상 철학**:
- 주색: 따뜻한 주황색(#FF8C42) - 에너지, 희망, 케어
- 보조색: 부드러운 핑크(#FFB6C1) - 부드러움, 아기
- 중립색: 크림색(#FFFAF0) - 따뜻함, 안정감
- 악센트: 자연 초록(#6B9E7F) - 성장, 자연, 회복

**레이아웃 패러다임**:
- 중앙 정렬 구조에 비대칭 일러스트레이션 배치
- 스토리 기반 섹션 흐름(문제→해결→결과)
- 원형 모티프로 순환적 구조 표현

**시그니처 요소**:
- 손으로 그린 듯한 부드러운 일러스트레이션
- 아기의 웃음과 부모의 안심 표현
- 따뜻한 톤의 배경 그래픽

**상호작용 철학**:
- 친근한 마이크로 인터랙션
- 터치/클릭 시 반응적인 피드백
- 스크롤 시 이야기가 펼쳐지는 느낌

**애니메이션**:
- 부드러운 바운스 효과
- 스크롤 시 요소의 부드러운 등장
- 호버 시 따뜻한 색상 변화

**타이포그래피**:
- 디스플레이: Playfair Display Bold(700) - 우아하고 따뜻한 헤딩
- 본문: Lato Regular(400) - 친근하고 읽기 편한 텍스트
- 강조: Lato SemiBold(600) - 부드러운 강조

### 3. 기술 혁신 포커스 (Tech-Forward Innovation)
**확률: 0.09**

**설계 철학**: AI와 기술 혁신을 강조하면서 의료 신뢰성을 동시에 전달

**핵심 원칙**:
- 현대적이고 미래지향적인 비주얼
- 기술 스택과 혁신 강조
- 데이터 기반의 신뢰성 표현
- 스타트업의 에너지와 역동성

**색상 철학**:
- 주색: 진한 청색(#1A1F71) - 기술, 신뢰, 전문성
- 보조색: 네온 핑크(#FF006E) - 에너지, 혁신
- 중립색: 어두운 회색(#2D2D2D) - 현대성
- 악센트: 네온 초록(#00F5FF) - 기술, 미래

**레이아웃 패러다임**:
- 그리드 기반 모듈식 구조
- 기술 스택 시각화
- 데이터 대시보드 스타일 요소

**시그니처 요소**:
- 기하학적 패턴과 선
- 디지털 파형 모티프
- 기술 데이터 시각화

**상호작용 철학**:
- 빠르고 반응적인 인터랙션
- 호버 시 기술적 효과
- 스크롤 시 동적 변화

**애니메이션**:
- 선형적이고 빠른 애니메이션
- 글리치 효과 활용
- 스크롤 시 패럴렉스 효과

**타이포그래피**:
- 디스플레이: Space Mono Bold(700) - 기술적, 모던
- 본문: IBM Plex Sans Regular(400) - 깔끔하고 기술적
- 강조: IBM Plex Sans SemiBold(600) - 강한 강조

## 선택된 디자인 철학

**"현대적 의료 기술 미니멀리즘"** 을 최종 선택합니다.

### 이유:
1. **의료 신뢰성**: 깔끔한 디자인으로 의료 전문성 표현
2. **감정적 연결**: 따뜻한 색상으로 부모의 불안감 완화
3. **스타트업 정체성**: 현대적이면서도 접근 가능한 이미지
4. **정보 명확성**: 미니멀한 구조로 핵심 메시지 전달
5. **확장성**: 향후 다양한 페이지와 기능 추가에 유리

### 디자인 가이드라인:
- **색상 팔레트**: 청색(#0066CC), 산호색(#FF6B6B), 밝은 회색(#F5F5F5), 초록색(#00D084)
- **타이포그래피**: Poppins(헤딩) + Inter(본문)
- **여백**: 넉넉한 패딩과 마진으로 호흡감 확보
- **일러스트레이션**: 아기의 움직임을 추상화한 곡선 모티프, 부드러운 원형 일러스트
- **상호작용**: 부드러운 호버 효과, 스크롤 트리거 애니메이션

1. **Hero 배경 이미지**: 아기와 부모의 따뜻한 순간을 표현하는 고품질 이미지
2. **About 섹션 이미지**: 의료 기술과 혁신을 시각화한 추상적 이미지
3. **제품 특징 일러스트레이션**: 스트레칭, AI 측정, 커뮤니티 등 주요 기능 표현
4. **팀 배경**: 전문성과 따뜻함을 동시에 표현하는 배경
5. **Contact 섹션 배경**: 연결과 소통을 표현하는 이미지
